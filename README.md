# OIBSIP_datascience_Task_1
# Iris Flower Classification

## Overview

This project implements a machine learning pipeline to classify Iris flowers into three species:

* Iris Setosa
* Iris Versicolor
* Iris Virginica

The classification is performed using flower sepal and petal measurements. Multiple machine learning models are trained, optimized, and compared to identify the best-performing classifier.

## Features

* Data preprocessing
* Feature engineering
* Feature scaling
* PCA visualization
* Hyperparameter tuning using GridSearchCV
* Cross-validation
* Model comparison
* Confusion Matrix visualization
* Feature importance analysis
* SHAP explainability

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn
* SHAP

## Models Implemented

* Logistic Regression
* Support Vector Machine (SVM)
* Random Forest Classifier
* Multi-Layer Perceptron (MLP)

## Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1-Score
* Cross Validation Accuracy

## Project Workflow

Dataset → Data Cleaning → Feature Engineering → Scaling → Model Training → Hyperparameter Tuning → Model Evaluation → Visualization → Explainability

## Results

The models were evaluated and compared using multiple performance metrics, and the best-performing model was selected based on overall classification performance.
